Question: were blocked on a simple problem:

what was the problem? I got really blocked at the end of the restructuring a string problem in the Javascript kata. I tried a lot of different solutions, including reading more about the method (.filter) I was using, and how it worked, restructuring my code out to show each step, finally I used the test code in MND docs to put in parts of my code and figured out the problem was a relatively simple space between .join method and .split method. I learnt how to refine my problem solving by using the MDN windows and the importance of small sections on how the final output works.
